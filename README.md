# SITIO WEB CON ANGULAR.JS NODE.JS

	- Sitio web creado para crear, leer, eliminar y editar contenido en  
	base de datos MySql con Symfony 2.7.  	 

# Herramientas:
	
	- Symfony 2.7
	- Php
	- MySql
	- Javascripy
	- Ajax
	- Bootstrap v3.1.5
	- jQuery v3.2.1
	- Font Awesome 4.3.0
	- html5 css3
